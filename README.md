# 🚀 pult-store-2.0
## 📖 О проекте 
pult-store-2.0 — современный интернет-магазин техники на базе стека MERN (MongoDB, Express, React, Node.js).
Приложение предоставляет удобный, адаптивный интерфейс для поиска, фильтрации и покупки товаров с безопасной авторизацией.

✨ Возможности
🛒 Каталог товаров с красочными изображениями и подробными описаниями

🔍 Поиск товаров по названию

💰 Фильтрация по диапазону цен

🛍️ Добавление товаров в корзину с учётом наличия

📦 Просмотр истории заказов и статусов

🔐 Аутентификация и авторизация пользователей через JWT

⚡ Быстрая и плавная клиентская часть на React

🛠 Технологии
React — фронтенд UI

Node.js + Express — сервер и REST API

MongoDB — база данных

Axios — HTTP-клиент для запросов

React Router — маршрутизация

CSS Grid & Flexbox — адаптивная вёрстка

⚙️ Установка и запуск
Клонируйте репозиторий:
bash
git clone https://github.com/artEvg/pult-store-2.0.git
Установите зависимости для сервера и клиента:

bash
cd server
npm install
cd ../client
npm install
Настройте .env файлы с переменными окружения (например, URL базы данных и порт).

Запустите сервер:

bash
npm run server
Запустите клиент:

bash
npm run dev
Откройте в браузере http://localhost:5173
